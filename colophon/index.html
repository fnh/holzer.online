Colophon

The source code of this website is available on github. To build the site, I use intertwingle, a node.js based static site generator that I've tailored to my own needs.

Hosting

The website is served via nginx which runs on a Ubuntu-based VPS hosted by IONOS (which is also my domain registrar).
